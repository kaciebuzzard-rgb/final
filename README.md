# Kacie Wielgus (Buzzard) — Personal Website

This repository contains the source code for **kacies.life**, the personal website of **Kacie Wielgus (Buzzard)** — speaker, writer, and strategist working at the intersection of disability advocacy, caregiving systems, and inclusive communications.

The site serves as a platform for:

- Speaking and consulting inquiries
- Selected writing and essays
- Excerpts from the memoir *The Waiting Rooms of My Life*
- Books and publications
- Advocacy work and thought leadership

---

## 🌐 Live Site

**https://kacies.life**

---

## ✨ Features

- Speaker and author homepage
- Memoir excerpt page
- Writing portfolio
- Books section
- Speaking and consulting information
- Contact and media inquiries

The site is designed to be **fast, accessible, and easy to update**.

---

## 🛠 Tech Stack

The site is built using:

- **Jekyll**
- **GitHub**
- **Netlify**
- **HTML / Markdown**
- **Custom CSS**

No JavaScript frameworks are used — the site is intentionally lightweight for performance and accessibility.
